# Server Configuration Parameter Sheets

このリポジトリは、**Rocky Linux 8** をベースとした汎用的なインフラサーバー構築における詳細なシステム設定値（パラメーターシート）を管理・ドキュメント化したものです。

構築時の構成管理、セキュリティ監査、およびトラブルシューティング時の参照用として活用することを目的としています。

## プロジェクトの目的

* **構成の可視化**: サーバー内部の複雑な設定を一覧化し、属人化を防止。
* **セキュリティ基準の準拠**: CISベンチマークに基づいた推奨値と実設定の比較。
* **保守性の向上**: 各項目の確認コマンドを併記し、実機での検証を容易にする。
