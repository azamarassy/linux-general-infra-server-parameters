# スワップ設定情報

| 項目 | デフォルト | 設定値 |
| --- | --- | --- |
| **swapファイル名** | 設定なし | `/swapfile` |
| **ブロックサイズ** | 設定なし | `1M` |
| **ブロック数** | 設定なし | `2048` (計2GB) |
| **Permission設定** | 設定なし | `600` |

---

### （参考）設定コマンド例

上記の設定値に基づいてスワップファイルを作成・有効化する手順は以下の通りです。

1. **スワップファイルの作成**
`dd if=/dev/zero of=/swapfile bs=1M count=2048`
2. **パーミッションの変更**
`chmod 600 /swapfile`
3. **スワップ領域のセットアップ**
`mkswap /swapfile`
4. **スワップの有効化**
`swapon /swapfile`
5. **永続化設定（/etc/fstabへの追記）**
`/swapfile swap swap defaults 0 0`
